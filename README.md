# Politician Trades

See what Congress is buying before the rest of the market does.

Track STOCK Act-disclosed trades from US politicians in seconds. Pull the 10 most recent trades for any member of Congress — ticker, side, size, and dates — rendered as a clean table inside Claude Code.

## Usage

```
/politician-trades
/politician-trades Tuberville
```

Triggers also fire on natural language: "what did Pelosi buy", "any new congress trades", "track [name]'s trades".

## Default

Pelosi (P000197). Pass any politician ID from Capitol Trades to switch.
